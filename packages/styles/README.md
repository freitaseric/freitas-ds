# @freitas-ds/styles

Camada CSS de compatibilidade do Freitas DS para consumidores existentes.

## Instalação

```bash
pnpm add @freitas-ds/styles tailwindcss
```

```css
@import "@freitas-ds/styles/index.css";
```

Novas aplicações com Tailwind CSS v4 devem preferir o plugin [`freitas-ds`](https://www.npmjs.com/package/freitas-ds).
